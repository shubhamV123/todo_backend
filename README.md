# todo_backend
A simple todo api.


## Installation
This is a [Node.js](https://nodejs.org/en/) based application.

Before installing, [download and install Node.js](https://nodejs.org/en/download/).
Node.js 0.10 or higher is required.

Once node installation is done.

Download or clone the project:

```bash
$ git clone https://github.com/shubhamV123/todo_backend.git
```
##### Note: change directory to order-management-backend before going further installation.



Install dependencies:

```bash
$ npm install
```
Start the server:

If you have installed nodemon then simply type:

```bash
$ nodemon
```
Find more information about nodemon [nodemon](https://github.com/remy/nodemon).

Or else simply type:

```bash
$ node index.js
```
